## How to Use These Sources

1. **Not as prescriptive medicine.** These are historical documents. Some recommendations are outdated or harmful by modern standards.
2. **As convergent evidence.** When an ancient text describes the same therapeutic use for a substance that modern RCTs confirm, that's a powerful signal — not because the ancients were infallible, but because 2,000 years of consistent observation across civilizations is data.
3. **With cultural respect.** These aren't "primitive guesses." They are the accumulated observations of sophisticated medical traditions that kept civilizations healthy for millennia.
