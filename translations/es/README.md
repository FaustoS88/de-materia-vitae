# De Materia Vitae — Spanish Translation

> Work in progress. Contributions welcome.

## How to Contribute

1. Copy a protocol file from `../protocols/`
2. Translate it to Spanish
3. Place in this directory, maintaining the same file name
4. Submit a PR

## Translation Guidelines

- Keep the Markdown formatting identical
- Keep ancient source quotes in their original language if available in Spanish translation; otherwise translate to Spanish
- Keep technical terms (PMID, DOI, evidence grade symbols) unchanged
- Use culturally appropriate examples where food items differ (e.g., suggest locally available berries, fish, etc.)
